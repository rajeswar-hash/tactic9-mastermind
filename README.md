# Tactic9 Mastermind

A strategic Tic-Tac-Toe inspired web game built using React and TypeScript.

## 🚀 Features
- Interactive 9-grid gameplay
- Smart game logic
- Responsive design
- Clean modern UI

## 🛠 Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS

## 📦 Installation

Clone the repository:

git clone https://github.com/rajeswar-hash/tactic9-mastermind.git

Install dependencies:

npm install

Start development server:

npm run dev
